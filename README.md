# 🚀 Real-Time Brand Guardian AI

An AI-powered system that monitors and analyzes brand-related content in real time, detects potential risks, and generates context-aware responses.

---


# 🔥 Key Features

- 🧠 Multimodal Analysis (Text + Image)  
- ⚡ Real-Time Threat Detection  
- 🤖 Gemini LLM Integration  
- 🗂️ Vector Memory (ChromaDB)  
- 🛡️ Automated Response Generation  

---

## 🏗️ Tech Stack

- **Backend:** FastAPI  
- **Frontend:** Streamlit  
- **AI:** Gemini API  
- **Database:** ChromaDB  
- **Language:** Python  

## 🔌 API
- POST /api/v1/analyze → Analyze content
- GET /health → Health check

---

## 💡 Use Cases
- Brand monitoring
- Fake news detection
- Customer feedback analysis
